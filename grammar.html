Grammar

1. Pronouns

1.1 Singular and Plural

Singular:

  1. ma = “I”
  2. ta = “you” (singular)
  3. ra = “he/she/it” (3rd person singular)

Plural:

  • Use z- prefix on pronouns:
    • zma = “we”
    • zta = “you (plural)”
    • zra = “they”

Note: Čeluria is an analytic language, so pronouns do not change form for case, gender, or politeness. There is no polite pronoun distinct from ta.

Example:
zma guo miru doka = “We saw the dog.”

1.2 Usage

  • No morphological changes for subject or object roles.
  • Negation or tense markers never attach to pronouns; they appear before verbs.

2. Nouns

2.1 No Case Inflection

  • Nouns remain the same in subject or object positions.
    • Example: doka (“dog”) is used identically whether the dog is doing something or having something done to it.

2.2 No Articles / Number

  • Čeluria has no “the” or “a.” To express “a dog,” say “one doka.”
  • Nouns do not change for plural. Plural context is typically indicated by quantifiers (“two doka”) or left to context.

2.3 Possession (“X de Y”)

  • de is used to indicate possession or association.
    • Example: ma de raku = “my book,” ra de vane = “his tree.”

3. Verbs

3.1 No Conjugation

  • Verbs never inflect for person or number.
  • Example: miru (“to see”) remains the same for “I see,” “you see,” or “they see.”

3.2 Tense/Aspect Markers

Five particles placed before the main verb:

  1. guo = past
    Example: ma guo miru noka = “I saw the cat.”
  2. wan = perfect/completed
    Example: zra wan gemu meke = “They have already eaten fish.”
  3. zai = progressive (ongoing)
    Example: ra zai fera ma = “He is speaking to me (right now).”
  4. chang = habitual/repeated
    Example: ta chang ču mive = “You (habitually) go to the house.”
  5. wei = future
    Example: zma wei shili raku = “We will write (a) book.”

Only one marker is used per clause. To express combined nuances, you can use multiple clauses or rely on context.

3.3 Negation

  • x is inserted immediately before the verb.
  • Example: ma x miru doka = “I do not see the dog.”
  • If used with a tense marker: ma x guo miru doka = “I did not see the dog.”

4. Adjectives

4.1 Position

  • Adjectives come before the noun.
  • Example: If “lifa” = “alive,” then lifa noka = “alive cat.”

4.2 No Agreement

  • Adjectives do not change form for number or gender.
  • If you have multiple adjectives, they proceed from left (broad) to right (specific). (You can expand on this with more adjectives later.)
  • (Your 100-word list has minimal explicit adjectives, e.g. “lifa” can function as “living, alive.”)

5. Adverbs

5.1 Derived with “-li”

  • Adverbs typically form by adding -li to an adjective root.
  • Example: “lifa” + “-li” = “lifa-li” = “in an alive manner (lively).”

5.2 Placement

  • Either before or after the verb.
  • Example: ma lifa-li miru doka or ma miru lifa-li doka → “I see the dog in a lively manner.”

6. Word Order & Syntax

6.1 Default SVO

  • Subject – Verb – Object is standard for clarity (nouns do not show case).
  • Example: ma miru doka = “I see the dog.”

6.2 Emphasis by Reordering

  • VSO: focus on the action → miru ma doka.
  • SOV: focus on the object → ma doka miru.
  • Context or pronouns clarify subject vs. object.

7. Question Formation

7.1 Yes/No Questions

  • Attach -ka to the main verb or place ka at the end.
    • ma miru-ka doka? = “Do I see the dog?”
    • ma miru doka ka? = same meaning.

7.2 Wh-Questions

  • Use question words (e.g. shui = “who,” shen = “what”), then -ka.
  • Example: shui guo miru noka ka? = “Who saw the cat?”
  • Čeluria’s 100-word set does not currently include “shui,” “shen,” but these can be added as function words.

8. Sample Sentences

Below are 10 examples, referencing basic words from your final 100-word set plus minimal function markers:

  1. ma miru doka
    SVO: “I see the dog.”
    ma = I, miru = see, doka = dog.
  2. ta x miru meke
    Negation: “You do not see the fish.”
    x = not.
  3. ra guo ču mive
    Past: “He went to the house.”
    guo = past, ču = go, mive = house.
  4. zma wan gemu meke
    Perfect: “We have eaten fish (already).”
    wan = completed aspect.
  5. zra wei miru noka
    Future: “They will see the cat.”
    wei = future.
  6. ma chang kera
    Habitual: “I habitually run.”
    chang = repeated action.
  7. ta miru-ka raku?
    Yes/no question: “Are you seeing the book?”
    -ka on the verb.
  8. ma x guo shili raku
    Negation + Past: “I did not write the book.”
    x = negation, guo = past, shili = write, raku = book.
  9. ra zai gemu meke
    Progressive: “He is eating fish.”
    zai = ongoing.
  10. zma lira-li tera
    (Hypothetical if “lira” is an adjective meaning “emotional,” then “lira-li” = “emotionally.”)
    Adverb usage: “We think emotionally.”
    tera = think, -li = adverb suffix.
